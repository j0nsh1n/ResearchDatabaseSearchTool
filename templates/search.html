Reading these results

The match badge shows how strong the similarity is (hover for the exact 0-1 score). A Type tag is a rough automated guess from the title and abstract (review, trial, survey, …). An exclamation mark means low or medium confidence - the tag is often wrong. It is not an evidence grade or quality score. Matching words from your query are highlighted in abstracts. Use ★ to bookmark and Add note for private notes.

Export these search results

Download the papers currently shown above as one RIS file for free Zotero (APA / MLA / Chicago after import). Only this ranked list — not your whole library.

Then format citations in Zotero (free)

  1. Click Download RIS (these results) above.
  2. Open the Zotero desktop app (install once if needed).
  3. In Zotero’s top menu bar (not a right‑click): choose File → Import….
  4. Select A file (BibTeX, RIS, Zotero RDF, etc.) → Next.
  5. Browse to the downloaded search_results.ris → Open → Next / Finish.
  6. You should now see many paper titles in your library — not a single row named “library” or “search_results” with a paperclip/file icon.
  7. Click the first paper, then Shift‑click the last to select them all (or Ctrl/Cmd‑A in the list). Right‑click → Create Bibliography from Items… → choose APA or MLA → copy into your document.

If you only see one item called “library” or “search_results”: you attached the file instead of importing it. Delete that row, then use File → Import… again — do not drag the file into the middle of the Zotero window.
