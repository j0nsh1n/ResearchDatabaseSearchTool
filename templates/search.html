Reading the results and exporting this list

The match badge shows how strong the similarity is (hover for the exact 0-1 score). A Type tag is a rough automated guess from the title and abstract (review, trial, survey, …). An exclamation mark means low or medium confidence - the tag is often wrong. It is not an evidence grade or quality score. Matching words from your query are highlighted in abstracts. Use ★ to bookmark and Add note for private notes.

These two buttons export only the current search results (the ranked list above), not your whole library:

  • CSV - opens in Excel or Google Sheets. Good for sorting, highlighting, or a screening table of this ranked set.
  • Text - plain reading list (easy to paste into a doc or print). Not for Zotero.

For the full collection or bibliography-manager files (RIS/BibTeX/APA), use Export your library below.

Assignment hand-in pack

One click downloads a .zip teachers often want: screening report counts, plus your included-only papers as CSV and RIS. Soft checklist below is hints only — it never blocks download.

What is inside the zip?
  • screening_report.txt — collected / excluded / included counts (process).
  • included_papers.csv — final set spreadsheet. Columns include Starred (★ bookmarks) and Note (private study notes) so teachers can see optional process evidence. Not a grade.
  • included_papers.ris — import into Zotero/EndNote for a real bibliography.
  • README_handin.txt — short explanation of each file.

Soft checklist (hints only)

Set optional targets your teacher mentioned. These never auto-fail or block export.

    Export your library

    Download your whole collection (or a scope you pick), not just the last search. Choose a format based on what you will do next. Prefer the Assignment hand-in pack above when a teacher wants process + final list together.

    Which file format should I pick?
    • CSV - spreadsheet of your library with cluster labels, screened-out status, Starred, and Note columns. Best for hand-in logs, teachers reviewing process, or working in Excel / Google Sheets. Not a bibliography style. Teachers: sort on Starred / Note to see which papers students marked for follow-up.
    • RIS (Zotero / EndNote / Mendeley) - standard citation exchange file. Import into a reference manager, then generate APA, MLA, Chicago, etc. there. Prefer this when you need a real bibliography database.
    • BibTeX - standard for LaTeX and many managers (including Zotero). Use for papers written in LaTeX or when a teacher asks for a .bib file.
    • APA (text) - plain-text draft in approximate APA 7th style for quick classroom reading lists. Author names and punctuation from databases can be imperfect - treat as a draft, not a final reference list. For polished styles, import RIS/BibTeX into Zotero or use ZoteroBib.

    Scope filters which papers go in the file:

    • All articles - everything you fetched in this library.
    • Included only - papers still in your final set (not screened out).
    • Screened out only - papers you excluded (duplicates, cluster triage, manual).
    • Starred only - papers you bookmarked with ★ (private; appears as the CSV Starred column when you export a wider scope).

    Need counts for a screening hand-in (how many collected / excluded / kept)? Use the hand-in pack above or the Screening report on the Duplicates page, not these citation formats alone.

    Tip: most students use RIS or BibTeX for a reference manager, and CSV when a teacher wants a screening spreadsheet with notes and stars.
