Your active shares

Class codes you created. Each student who joins gets a clone (their own copy), not live access to yours. Revoke if a code was posted publicly by mistake — students who already joined keep their copy.

What to tell students after you share
  1. Share the code (or /join?code=… link) and the library name/unit.
  2. Students: Join → open the new library in the nav switcher.
  3. Suggested next steps: screen on Clusters → clean duplicates → Search & export RIS; optional screening report on Duplicates for process counts.
  4. Remind them: public databases only (starting point), not a full library search.
